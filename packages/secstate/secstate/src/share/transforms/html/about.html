About SecState
SecState is a tool that attempts to streamline the Certification and Accreditation (C&A) process of Linux systems. SecState provides a mechanism to verify, validate, and provide remediation to security relevant configuration items. The projects main goals are to provide a user friendly interface to configure the security state of a system and generate evidence that it is configured as such, to be usede within the C&A process.
